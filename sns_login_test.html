SNS 로그인 테스트

현재 Rails 서버가 실행되지 않아 실제 로그인은 불가능합니다.
하지만 OAuth URL 구조를 확인할 수 있습니다.

SNS 로그인 버튼
Google Kakao Naver

환경 변수 설정 상태

✅ Google OAuth: 설정 완료

✅ Kakao Login: 설정 완료

✅ Naver Login: 설정 완료

서버 실행 방법

1. Docker 방식: ./simple-start.sh

2. 직접 실행: rails server

3. 클라우드 배포 (Heroku, Railway 등)

구현된 기능

✅ OAuth 콜백 핸들러

✅ 사용자 자동 생성

✅ 구독 티어 관리

✅ API 사용량 추적
